# Celgene (celgene)

Celgene Corporation was a global biopharmaceutical company focused on innovative therapies for cancer and immune-inflammatory diseases. Celgene was acquired by Bristol Myers Squibb (BMS) in November 2019. This repository captures the APIs, developer tools, and machine-readable API artifacts for Celgene.
